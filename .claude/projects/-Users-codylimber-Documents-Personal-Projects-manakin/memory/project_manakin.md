Manakin is a field companion app for species phenology using iNaturalist community science data. Shows which species are active near you, with data packs, targets, trip planning, widgets, and notifications.

**Why:** Personal project being prepared for a larger public release (as of 2026-04-28).

**How to apply:** Focus on production-readiness, polish, and robustness when reviewing or modifying code.
